# SFC Scannow

## 개요
이 명령어는 현재 실행 중인 Windows 시스템 파일의 무결성을 검사하고 손상된 파일을 복구할 때 사용하는 복구 명령어입니다.

보호된 Windows 시스템 파일의 손상 여부를 확인하고 복구하는 데 사용됩니다.

## 문법
```cmd
sfc /scannow
```

## 옵션
- `sfc`  
  System File Checker의 약자로, Windows 시스템 파일의 무결성을 검사하고 복구에 사용하는 명령줄 도구입니다.

- `/scannow`  
  보호된 시스템 파일 전체를 즉시 검사하고, 손상된 파일이 있으면 복구를 수행합니다.

## 방법

### 1. 관리자 권한으로 실행
시작 메뉴에서 `명령 프롬프트` 또는 `PowerShell`을 검색한 뒤 관리자 권한으로 실행합니다.

### 2. 명령어 입력
아래 명령어를 입력하고 실행합니다.

```cmd
sfc /scannow
```

## 결과
실행 후 현재 Windows 시스템 파일의 손상 여부와 복구 결과를 확인할 수 있습니다.

## 참고
- 관리자 권한으로 실행하는 것이 좋습니다.
- 점검 또는 복구 작업 전에는 중요한 데이터 백업을 권장합니다.
- `sfc /scannow`는 시스템 파일 전체를 검사하므로, 시스템 상태에 따라 실행 시간이 달라질 수 있습니다.
