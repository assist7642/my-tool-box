## 📖 상세 설명

### 1. Time Fixer
- **문제 인식**: UI 테스트나 튜토리얼 캡처 시 시계가 계속 변하는 불편함 해결
- **최적화**: `timeout`을 사용해 CPU 부하를 방지한 Stable 버전 제공
- **편의성**: VBScript를 통해 백그라운드 실행 지원
